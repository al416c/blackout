Refonte : Évolution par Terminal

Le terminal devient l'outil d'achat unique. L'arbre sert de guide visuel.

Arbre d'Évolution (Visuel)
[TRANSMISSION]
Scan réseau (N1)
> cmd: scan.deploy --force
[TRANSMISSION]
Exploitation SMB (N2)
> cmd: exploit.smb --target=all

REQUIS : 120 CPU
↓
Exploit Zero-Day (N3)
Vérifiez les prérequis...
Terminal (Action)
root@blackout:~# exploit.smb --target=all
[WAIT] Initialisation du module SMB...
[INFO] Coût : 120 CPU Cycles
[PROG] Injection : [==========----------] 50%
[WARN] Bruit réseau généré : +8% méfiance

Pourquoi cette approche ?

Avantages

  • Cohérence thématique totale (on ne clique pas, on hack).
  • Courbe d'apprentissage gratifiante (on finit par connaître ses commandes).
  • Possibilité d'ajouter des drapeaux (flags) aux commandes pour varier les effets.

Défis

  • Nécessite une aide visuelle claire pour ne pas frustrer le joueur.
  • Gestion des fautes de frappe (ajouter une auto-complétion ?).
